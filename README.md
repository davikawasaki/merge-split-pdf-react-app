# Merge & Split PDF React App

Simple ReactJS serveless application that merges and splits PDFs.

## Dependencies

- @material-ui/core (latest)
- pdf-lib (^0.6.1)
- prop-types: (latest)
- react: (latest)
- react-dom: (latest)
- react-files: (^2.4.8)
- react-scripts: (latest)
- save-file: (^2.3.1)

## Install dependencies, running local server and building

```sh
$ npm i
$ npm run start
$ npm run build
```
